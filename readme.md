FLEET MONITORING & PREDICTIVE MAINTANENCE USING SAGEMAKER :
A real-time fleet monitoring system built using AWS services,including S3, Lambda, SageMaker, DynamoDB, SNS, and CloudWatch.

FEATURES :
- Real-time monitoring from S3
- Lambda data processing
- SageMaker predictions
- DynamoDB storage
- SNS alerts for critical temperature
